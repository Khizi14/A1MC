Goal of the Final App

The goal of the final application is to be user-friendly were a user can have appointment with a doctor through app and talk to doctor on a chat.

Main Goal

  1. Fast and easy way to get doctor appointment.

  2. No physical need to go to doctor office and appointment can be done on application.

  3. Different type of doctors for different issues (Doctors List).

  4. Search bar to look for specific doctor.

  5. Talk to Doctors in chat and get healty.

Sub Goal

  1. User can upload profile picture.

  2. send images via chat.

Due to Covid 19 still in effect in few contries having a doctor access via chat can safe many lifes.
